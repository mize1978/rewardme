# Mize Tasks 🍰

ごほうびでタスクを続けるシンプルなToDoアプリです。

## URL
https://reward-task-app.onrender.com/

## GitHub
https://github.com/mize1978/reward_task_app

## 機能
- タスク作成
- タスク一覧表示
- タスク完了 / 未完了切り替え
- タスク削除

## 技術
- Ruby on Rails
- PostgreSQL
- Render

## こだわり
タスクを「やること」ではなく「ごほうび」にすることで、
楽しく続けられる体験を意識しました。
